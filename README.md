# JACINTO

Portfolio website for a boutique plant shop.

## Overview

JACINTO is a one-page plant store concept with an editorial hero, custom botanical logo, ceramic-inspired visual system, shop photography, and a curated plant catalog.

## Commands

```bash
npm install
npm run dev
npm run build
```

## Stack

- Next-style app structure through Vinext
- React
- Tailwind entry with custom CSS
- Static image catalog in `public/assets`
